# 프로토타입 와이어프레임

## 캔버스 레이아웃

````

+----------------------------+
\| \[언어 선택 ▼] \[저장] \[내보내기] |
+----------------------------+
\|                            |
\|   \[노드]   →   \[노드]       |
\|     ↓           ↘         |
\|   \[선택지 노드]            |
\|                            |
+----------------------------+

```

---

## 노드 상세 예시

### text 노드

```

┌──────────────┐
│ 제목: 첫 대사   │
├──────────────┤
│ 화자: turtle1 │
│ 대사: key\_001 │
│ 미리보기: 안녕! │
├──────────────┤
│ \[다음 노드 연결]│
└──────────────┘

```

### choice 노드

```

┌──────────────┐
│ 제목: 선택지   │
├──────────────┤
│ 화자: turtle2 │
│ 대사: key\_002 │
│ 미리보기: 어떻게 생각해? │
├──────────────┤
│ - 선택지1: key\_101 → 노드 A │
│ - 선택지2: key\_102 → 노드 B │
└──────────────┘

```
